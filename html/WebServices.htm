SCDJWS: Sun Certification for Web Services

Even though web services has been around for several years, it has caught a great attention after "SOA" making buzz around. The definition of SOA-Service Oriented Architecture is not limited to Web Services but you will hardly see any implementation of SOA without using web services one or the other way.

For those planning to appear for Sun Certification for Java Web Services, here are some useful review notes:

  • Objectives with difficult ones highlighted:  http://docs.google.com/Doc?docid=d77kkqc_9zcmtpqfc&hl=en
  • Preparation plan template for SCDJWS. It also consists of list of reference books and links for SCDJWS. Good reading for web services in general: http://spreadsheets.google.com/pub?key=p1ZX99bB0jPhYUPR8dnqYKA
  • SCDJWS Exam Cram : http://docs.google.com/Doc?docid=d77kkqc_7fh5wj4&hl=en
  • SCDJWS points in tabular format: http://docs.google.com/Doc?docid=d77kkqc_5ggjkmd&hl=en
  • Web services useful code snippets: http://docs.google.com/Doc?docid=d77kkqc_6t2nhch&hl=en
  • Notes from Basic Profile 1.0: http://docs.google.com/Doc?docid=d77kkqc_13hpzb87dm&hl=en
